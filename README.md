# config-files

## Description

This repository contains configuration files for various tools and applications


## Bash Config

```shell
~/.bashrc

```

## Tmux Config

```shell

    ~/.tmux.conf

```

## Vim Config

```shell
    ~/.vimrc
    ~/.ideavimrc
```

## Nvim script config

```shell
    ~/.config/nvim/init.vim
```

## Nvim Lua Config 

```shell
    ~/.config/nvim/init.lua
    ~/.config/nvim/lua/config
    ~/.config/nvim/lua/plugins
```
